# App Privacy Details Draft

Use this draft in App Store Connect after legal review.

## Data Collected

- Contact Info: email address for account login and support.
- User Content: project names, progress details, purchase/source notes, idea details, supply inventory notes, optional chat requests, community reports, and shared community posts.
- Identifiers: internal user ID and authentication token.
- Sensitive Info: date of birth and gender for account safety and personalization.
- Location: optional user-entered location only when location sharing is enabled.

## Data Use

- App Functionality: account access, project tracking, idea details, inventory notes, optional chat requests, settings, moderation, and community safety.
- Personalization: connection range and optional profile preferences.
- Other Purposes: support and abuse prevention.

## Linked To User

Email, profile fields, saved projects, ideas, inventory, optional chat requests, reports, blocks, and optional location are linked to the user account.

## Tracking

No cross-app tracking is implemented in this build.

## Third-Party SDKs

Capacitor is used as the native shell. No analytics, advertising, or crash-reporting SDK is configured in this build.
