# App Store Release Checklist

## Ready In This Project

- Mobile and tablet app layout.
- Capacitor iOS wrapper configuration.
- App metadata draft.
- Privacy policy, terms, support, and community guidelines pages.
- Account deletion route and Settings action.
- Report and block controls for community content.
- Optional location sharing.
- Optional community chat requests.
- First-time date of birth and gender profile setup.
- Editable projects and ideas.
- Practical inventory notes for kits, waxes, magnets, trays, pens, sealer, and optional drill codes.
- Demo account information for App Review.

## Required Before Submission

- Apple Developer Program membership.
- Xcode signing team and provisioning profile.
- Production API hosted at `https://api.prismpatch.app/api`.
- Production database, backups, and moderation dashboard.
- Moderation workflow for reports and chat abuse.
- Real support and privacy-policy URLs.
- Legal review of Privacy Policy and Terms of Use.
- Final app icons and screenshots uploaded in App Store Connect.
- On-device testing on iPhone and iPad.
- TestFlight build review before public submission.
- Upgrade Node and Capacitor to remove current CLI audit warnings before release.
