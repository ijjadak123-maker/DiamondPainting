# App Review Notes

What to review:

- First-time profile setup asks for date of birth and gender.
- Location sharing is optional and off by default.
- Home shows current and possible next projects.
- Projects can be saved and edited with progress percent or completed/total sections.
- Ideas can be opened for details and edited.
- Inventory tracks not-started kits, waxes, magnets, trays, pens, sealer, optional drill color codes, and free-form notes.
- Community gallery posts show owner progress, where the project was bought, time worked, notes, Report and Block controls.
- Optional community chat requests are controlled by the Settings opt-in.
- Settings includes Privacy Policy, Terms, Community Guidelines, Support, Sign out, and Delete account.

- API base URL: `https://api.prismpatch.app/api`
- Health check: `https://api.prismpatch.app/api/health`
